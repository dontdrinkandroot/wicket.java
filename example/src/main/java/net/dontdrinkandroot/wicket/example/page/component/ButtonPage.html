ButtonBehavior

The ButtonBehavior is a Behavior that manages the styling of components that should be shown as button.

Styles

Button Styles are encapsulated in the ButtonStyle Enumeration

Available Styles

Example Code

link.add(new ButtonBehavior(ButtonStyle.PRIMARY));
link.add(new ButtonBehavior().setButtonStyle(ButtonStyle.PRIMARY));

Sizes

Button Sizes are encapsulated in the ButtonSize Enumeration

Available Sizes

Example Code

link.add(new ButtonBehavior(ButtonSize.LARGE));
link.add(new ButtonBehavior().setButtonSize(ButtonSize.LARGE));
